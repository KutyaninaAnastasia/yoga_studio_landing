Студия йоги и фитнеса

NAME

Уютное пространство для женщин

о студии

Йога - это возможность расслабиться, подумать о важном, почувствовать прилив сил и бодрости.

Ежедневно у нас проходит несколько классов йоги для различного уровня, есть классы и для начинающих и для продолжающих. Классы Йоги для начинающих включают практику Хатха йоги, пранаямы (дыхательные практики), энергетические практики, практики для раскрытия женственности и многое другое. Помимо этого мы предлагаем классы для беременных, танцы для детей, фитнесс, классы для женщин преклонного возраста.
